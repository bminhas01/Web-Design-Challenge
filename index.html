Summary - Weather Analysis


This project was designed to study the relationship between the location of a particular city and its climate. In order to remove potential bias in the selection process, 500+ cities were selected on random using an existing Python library (citypy). Weather data for the selected cities was pulled using the OpenWeather Map API on February, 9th 2021.

Once assembled, the dataset was used to create several plots using Matplotlib. These plots focused on factors such as maximum temperature, cloudiness, humidity, and wind speed, and compared them to the given city's latitude. The goal being to display trends across the globe and identify the correlation between the latitude and a given factor. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
